# Dailies Calendar

Editor action `roam.dailies_calendar`.

Open the month-grid calendar in dailies mode; pressing Enter on a day opens or creates that day's Org-roam daily note instead of inserting the date string.

From **Org -> Roam -> Dailies -> Calendar...**. `App::calendar_dailies` flag; `App::calendar_accept` routes Enter to `roam_open_daily`. See `crates/vix-org/spec/index.md`.

See `spec/index/index.md` for the project overview and `crates/vix-editor-core/spec/index.md` for the full action catalog.
