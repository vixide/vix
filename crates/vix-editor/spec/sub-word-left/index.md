# Sub Word Left

Editor action `sub-word-left`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `sub-word-left` |
| snake | `sub_word_left` |
| Pascal | `SubWordLeft` |

Run it from the command palette or a key binding via the action id `sub_word_left`.
It is dispatched by `App::run_action("sub_word_left")` and, for editing actions, backed
by `Editor::sub_word_left` in `editor_core`. See `crates/vix-editor-core/spec/index.md` for the full
catalog.
