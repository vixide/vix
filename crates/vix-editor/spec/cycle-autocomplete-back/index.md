# Cycle Autocomplete Back

Editor action `cycle-autocomplete-back`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `cycle-autocomplete-back` |
| snake | `cycle_autocomplete_back` |
| Pascal | `CycleAutocompleteBack` |

Run it from the command palette or a key binding via the action id `cycle_autocomplete_back`.
It is dispatched by `App::run_action("cycle_autocomplete_back")` and, for editing actions, backed
by `Editor::cycle_autocomplete_back` in `editor_core`. See `crates/vix-editor-core/spec/index.md` for the full
catalog.
