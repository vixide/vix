# Delete Sub Word Left

Editor action `delete-sub-word-left`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `delete-sub-word-left` |
| snake | `delete_sub_word_left` |
| Pascal | `DeleteSubWordLeft` |

Run it from the command palette or a key binding via the action id `delete_sub_word_left`.
It is dispatched by `App::run_action("delete_sub_word_left")` and, for editing actions, backed
by `Editor::delete_sub_word_left` in `editor_core`. See `crates/vix-editor-core/spec/index.md` for the full
catalog.
