# Indent Guides

Rendered decoration (no toggle action).

Draw a faint vertical bar at each indent level of space-indented lines, making block structure easy to follow.

`editor_core::render::draw_indent_guides`. See `crates/vix-editor/spec/index.md`.

See `spec/index/index.md` for the project overview and `crates/vix-editor-core/spec/index.md` for the full action catalog.
