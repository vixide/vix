# Right click

Right click on text displays a context menu with menuitems that are clipboard verbs:

| Label | Action |
|-|-|
| Cut | cut |
| Copy | copy |
| Paste | paste |
| *separator* ||
| Select All | select_all |
| Select More | select_more (use the selection otherwise select the word)
| Select Less | select_less (use the selection otherwise noop)
| *separator* |
| Find... | find with the selection |
| Find Next | find_next (use the selection otherwise use the word)
| Find Previous | find_previous (use the selection otherwise use the word)
