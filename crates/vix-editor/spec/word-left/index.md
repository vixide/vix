# Word Left

Editor action `word-left`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `word-left` |
| snake | `word_left` |
| Pascal | `WordLeft` |

Run it from the command palette or a key binding via the action id `word_left`.
It is dispatched by `App::run_action("word_left")` and, for editing actions, backed
by `Editor::word_left` in `editor_core`. See `crates/vix-editor-core/spec/index.md` for the full
catalog.
