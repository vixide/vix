# Start

Editor action `start`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `start` |
| snake | `start` |
| Pascal | `Start` |

Run it from the command palette or a key binding via the action id `start`.
It is dispatched by `App::run_action("start")` and, for editing actions, backed
by `Editor::start` in `editor_core`. See `crates/vix-editor-core/spec/index.md` for the full
catalog.
