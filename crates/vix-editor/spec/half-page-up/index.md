# Half Page Up

Editor action `half-page-up`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `half-page-up` |
| snake | `half_page_up` |
| Pascal | `HalfPageUp` |

Run it from the command palette or a key binding via the action id `half_page_up`.
It is dispatched by `App::run_action("half_page_up")` and, for editing actions, backed
by `Editor::half_page_up` in `editor_core`. See `crates/vix-editor-core/spec/index.md` for the full
catalog.
