# Scratch Buffer

Editor action `file.scratch`.

Open a throwaway, unsaved buffer (no file path) seeded with a header line, for quick notes or calculations.

From **File -> New Scratch Buffer** or the command palette. `App::new_scratch_buffer`.

See `spec/index/index.md` for the project overview and `crates/vix-editor-core/spec/index.md` for the full action catalog.
