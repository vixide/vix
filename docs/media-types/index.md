# Media Types

Look up and insert **media types** (MIME / content types) — like `image/png` or
`application/json` — without leaving the editor.

## Using the picker

Open **Tools → Media Types** (`tools.media_types`). You get a searchable list of
common media types, each with its description and file extension(s). If the file
you're editing has a recognized extension, the list opens pre-selected to its
media type.

- **Type** to filter by media type, description, or extension (e.g. `svg`, `font`,
  `.docx`).
- **↑/↓** and **PgUp/PgDn** move the highlight; **Backspace** edits the filter.
- **Enter** or a **click** inserts the highlighted media type at the cursor and
  keeps the panel open so you can insert several; **Esc** closes it.

The catalog covers the common web types plus popular developer and modern formats
(source code, config, archives, fonts, images, audio, and video). It is a curated
*common* set, not the full IANA registry.

See the specification and data table at
[`spec/media-types/`](../../spec/media-types/index.md).
