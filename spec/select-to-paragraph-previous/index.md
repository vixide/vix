# Select To Paragraph Previous

Editor action `select-to-paragraph-previous`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `select-to-paragraph-previous` |
| snake | `select_to_paragraph_previous` |
| Pascal | `SelectToParagraphPrevious` |

Run it from the command palette or a key binding via the action id `select_to_paragraph_previous`.
It is dispatched by `App::run_action("select_to_paragraph_previous")` and, for editing actions, backed
by `Editor::select_to_paragraph_previous` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
