# Select To Start Of Text Toggle

Editor action `select-to-start-of-text-toggle`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `select-to-start-of-text-toggle` |
| snake | `select_to_start_of_text_toggle` |
| Pascal | `SelectToStartOfTextToggle` |

Run it from the command palette or a key binding via the action id `select_to_start_of_text_toggle`.
It is dispatched by `App::run_action("select_to_start_of_text_toggle")` and, for editing actions, backed
by `Editor::select_to_start_of_text_toggle` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
