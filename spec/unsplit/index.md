# Unsplit

Editor action `unsplit`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `unsplit` |
| snake | `unsplit` |
| Pascal | `Unsplit` |

Run it from the command palette or a key binding via the action id `unsplit`.
It is dispatched by `App::run_action("unsplit")` and, for editing actions, backed
by `Editor::unsplit` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
