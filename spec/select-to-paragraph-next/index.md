# Select To Paragraph Next

Editor action `select-to-paragraph-next`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `select-to-paragraph-next` |
| snake | `select_to_paragraph_next` |
| Pascal | `SelectToParagraphNext` |

Run it from the command palette or a key binding via the action id `select_to_paragraph_next`.
It is dispatched by `App::run_action("select_to_paragraph_next")` and, for editing actions, backed
by `Editor::select_to_paragraph_next` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
