# Live Backlinks

Editor action `roam.backlinks_follow`.

A toggle that fills the bottom dock with the active Org-roam node's linked and unlinked references and refreshes as you move between nodes (keyed by active tab + revision).

From **Org -> Roam -> Live Backlinks**. `App::refresh_backlinks_follow`. See `spec/org/index.md`.

See `spec/index/index.md` for the project overview and `spec/actions/index.md` for the full action catalog.
