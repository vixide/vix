# Select Right

Editor action `select-right`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `select-right` |
| snake | `select_right` |
| Pascal | `SelectRight` |

Run it from the command palette or a key binding via the action id `select_right`.
It is dispatched by `App::run_action("select_right")` and, for editing actions, backed
by `Editor::select_right` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
