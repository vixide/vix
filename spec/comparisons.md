# Comparisons

Simpler than Fresh Editor (~/github/sinelaw/fresh) because:
- No plugins
- No JavaScript
- No TypeScript
- No remote editing
- No themes
- No git integration

More advanced than Micro Editor (~/github/micro-editor-micro) because:
- Left-side drawer is a file explorer
- Right-side drawer is a message explorer

Easier for novices than vim because:
- Keyboard shortcuts are typical of macOS and Windows, for example Ctrl-C for Copy.
- Not modal.

Easier for novices than emacs because:
- Keyboard shortcuts are typical of macOS and Windows, for example Ctrl-C for Copy.
- Menu bar at top is on by default.
- Command names are typical of macOS and Windows, for example Copy/Paste, not Kill/Yank.
