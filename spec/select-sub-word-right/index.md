# Select Sub Word Right

Editor action `select-sub-word-right`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `select-sub-word-right` |
| snake | `select_sub_word_right` |
| Pascal | `SelectSubWordRight` |

Run it from the command palette or a key binding via the action id `select_sub_word_right`.
It is dispatched by `App::run_action("select_sub_word_right")` and, for editing actions, backed
by `Editor::select_sub_word_right` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
