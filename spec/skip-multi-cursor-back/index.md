# Skip Multi Cursor Back

Editor action `skip-multi-cursor-back`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `skip-multi-cursor-back` |
| snake | `skip_multi_cursor_back` |
| Pascal | `SkipMultiCursorBack` |

Run it from the command palette or a key binding via the action id `skip_multi_cursor_back`.
It is dispatched by `App::run_action("skip_multi_cursor_back")` and, for editing actions, backed
by `Editor::skip_multi_cursor_back` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
