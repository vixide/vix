# First Split

Editor action `first-split`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `first-split` |
| snake | `first_split` |
| Pascal | `FirstSplit` |

Run it from the command palette or a key binding via the action id `first_split`.
It is dispatched by `App::run_action("first_split")` and, for editing actions, backed
by `Editor::first_split` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
