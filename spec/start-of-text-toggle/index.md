# Start Of Text Toggle

Editor action `start-of-text-toggle`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `start-of-text-toggle` |
| snake | `start_of_text_toggle` |
| Pascal | `StartOfTextToggle` |

Run it from the command palette or a key binding via the action id `start_of_text_toggle`.
It is dispatched by `App::run_action("start_of_text_toggle")` and, for editing actions, backed
by `Editor::start_of_text_toggle` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
