# Start Of Text

Editor action `start-of-text`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `start-of-text` |
| snake | `start_of_text` |
| Pascal | `StartOfText` |

Run it from the command palette or a key binding via the action id `start_of_text`.
It is dispatched by `App::run_action("start_of_text")` and, for editing actions, backed
by `Editor::start_of_text` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
