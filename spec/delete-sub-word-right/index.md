# Delete Sub Word Right

Editor action `delete-sub-word-right`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `delete-sub-word-right` |
| snake | `delete_sub_word_right` |
| Pascal | `DeleteSubWordRight` |

Run it from the command palette or a key binding via the action id `delete_sub_word_right`.
It is dispatched by `App::run_action("delete_sub_word_right")` and, for editing actions, backed
by `Editor::delete_sub_word_right` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
