# Center

Editor action `center`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `center` |
| snake | `center` |
| Pascal | `Center` |

Run it from the command palette or a key binding via the action id `center`.
It is dispatched by `App::run_action("center")` and, for editing actions, backed
by `Editor::center` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
