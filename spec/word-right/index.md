# Word Right

Editor action `word-right`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `word-right` |
| snake | `word_right` |
| Pascal | `WordRight` |

Run it from the command palette or a key binding via the action id `word_right`.
It is dispatched by `App::run_action("word_right")` and, for editing actions, backed
by `Editor::word_right` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
