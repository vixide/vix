# None

Editor action `none`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `none` |
| snake | `none` |
| Pascal | `None` |

Run it from the command palette or a key binding via the action id `none`.
It is dispatched by `App::run_action("none")` and, for editing actions, backed
by `Editor::none` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
