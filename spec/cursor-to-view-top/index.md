# Cursor To View Top

Editor action `cursor-to-view-top`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `cursor-to-view-top` |
| snake | `cursor_to_view_top` |
| Pascal | `CursorToViewTop` |

Run it from the command palette or a key binding via the action id `cursor_to_view_top`.
It is dispatched by `App::run_action("cursor_to_view_top")` and, for editing actions, backed
by `Editor::cursor_to_view_top` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
