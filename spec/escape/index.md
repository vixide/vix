# Escape

Editor action `escape`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `escape` |
| snake | `escape` |
| Pascal | `Escape` |

Run it from the command palette or a key binding via the action id `escape`.
It is dispatched by `App::run_action("escape")` and, for editing actions, backed
by `Editor::escape` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
