# Cursor Right

Editor action `cursor-right`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `cursor-right` |
| snake | `cursor_right` |
| Pascal | `CursorRight` |

Run it from the command palette or a key binding via the action id `cursor_right`.
It is dispatched by `App::run_action("cursor_right")` and, for editing actions, backed
by `Editor::cursor_right` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
