# Cursor To View Bottom

Editor action `cursor-to-view-bottom`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `cursor-to-view-bottom` |
| snake | `cursor_to_view_bottom` |
| Pascal | `CursorToViewBottom` |

Run it from the command palette or a key binding via the action id `cursor_to_view_bottom`.
It is dispatched by `App::run_action("cursor_to_view_bottom")` and, for editing actions, backed
by `Editor::cursor_to_view_bottom` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
