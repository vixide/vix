# Paste Primary

Editor action `paste-primary`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `paste-primary` |
| snake | `paste_primary` |
| Pascal | `PastePrimary` |

Run it from the command palette or a key binding via the action id `paste_primary`.
It is dispatched by `App::run_action("paste_primary")` and, for editing actions, backed
by `Editor::paste_primary` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
