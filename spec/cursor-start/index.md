# Cursor Start

Editor action `cursor-start`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `cursor-start` |
| snake | `cursor_start` |
| Pascal | `CursorStart` |

Run it from the command palette or a key binding via the action id `cursor_start`.
It is dispatched by `App::run_action("cursor_start")` and, for editing actions, backed
by `Editor::cursor_start` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
