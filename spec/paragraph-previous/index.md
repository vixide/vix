# Paragraph Previous

Editor action `paragraph-previous`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `paragraph-previous` |
| snake | `paragraph_previous` |
| Pascal | `ParagraphPrevious` |

Run it from the command palette or a key binding via the action id `paragraph_previous`.
It is dispatched by `App::run_action("paragraph_previous")` and, for editing actions, backed
by `Editor::paragraph_previous` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
