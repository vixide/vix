# Select Sub Word Left

Editor action `select-sub-word-left`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `select-sub-word-left` |
| snake | `select_sub_word_left` |
| Pascal | `SelectSubWordLeft` |

Run it from the command palette or a key binding via the action id `select_sub_word_left`.
It is dispatched by `App::run_action("select_sub_word_left")` and, for editing actions, backed
by `Editor::select_sub_word_left` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
