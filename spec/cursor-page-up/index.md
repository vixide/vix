# Cursor Page Up

Editor action `cursor-page-up`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `cursor-page-up` |
| snake | `cursor_page_up` |
| Pascal | `CursorPageUp` |

Run it from the command palette or a key binding via the action id `cursor_page_up`.
It is dispatched by `App::run_action("cursor_page_up")` and, for editing actions, backed
by `Editor::cursor_page_up` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
