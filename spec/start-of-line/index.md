# Start Of Line

Editor action `start-of-line`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `start-of-line` |
| snake | `start_of_line` |
| Pascal | `StartOfLine` |

Run it from the command palette or a key binding via the action id `start_of_line`.
It is dispatched by `App::run_action("start_of_line")` and, for editing actions, backed
by `Editor::start_of_line` in `vix-editor`. See `spec/actions/index.md` for the full
catalog.
