# Duplicate

Editor action `duplicate`.

| Form | Identifier |
| ---- | ---------- |
| kebab | `duplicate` |
| snake | `duplicate` |
| Pascal | `Duplicate` |

Run it from the command palette or a key binding via the action id `duplicate`.
It is dispatched by `App::run_action("duplicate")` and, for editing actions, backed
by `Editor::duplicate` in `editor_core`. See `spec/actions/index.md` for the full
catalog.
