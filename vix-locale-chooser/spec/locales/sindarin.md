# Locales: Sindarin

Sindarin is an imaginary language created by J.R.R. Tolkien in the books "The Lord of the Rings". Sindarin is one of the languages spoken by the Elf people.

The formal locale for the Sindarin language is identified by the ISO 639-3 code sjn. [1]

To form a complete BCP 47 language tag for programming or web localization, you combine this language code with optional script or region subtags: [2]

## Common Sindarin Locales

- Basic Sindarin: sjn
- Sindarin in Tengwar script: sjn-Teng (using the ISO 15924 script code Teng)
- Sindarin in Latin script: sjn-Latn (often used for transcriptions found in The Lord of the Rings appendices) [3, 4]

## Key Technical Details

- Language Type: Unlike Klingon (tlh), which is often listed in major translation tools, Sindarin is primarily used in niche linguistic libraries and custom localization layers. [5, 6, 7]
- Script Support: The native script, Tengwar, is not yet standardized in Unicode but is often mapped using the [ConScript Unicode Registry (CSUR)](https://en.wiktionary.org/wiki/Appendix:Tengwar). [3, 8]
- Regional Modifier: Since Sindarin is a fictional language, there is no official ISO 3166 region code for Middle-earth. Developers often use private-use tags like sjn-x-middle-earth or simply sjn. [5, 9, 10, 11]
