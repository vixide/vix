# Locales: Klingon

The locale for the Klingon language (tlhIngan Hol) is formally defined by the ISO 639-1/3 language code tlh.

While a standard regional modifier doesn't exist for the fictional planet of Qo'noS, systems that support custom or esoteric locales typically format it as tlh-Klingon or tlh-x-klingon.

Because Klingon is a constructed language (created by linguist Marc Okrand), full operating system locales aren't natively packaged by default.

However, it is widely utilized and recognizable in programming, translation, and text-processing tools:

- Language Code: tlhScript Code (pIqaD): Piqd (commonly used in Unicode mapping for the Klingon writing system)

- Common Identifier: tlh-x-klingon (used as an experimental or custom locale string in many localized apps.

- Using tlh is the universally accepted standard for the Klingon language.

Klingon (tlhIngan Hol) uses a strictly base-10 (decimal) system for numbers and specific suffixes for time units like years, months, and days. [1, 2]

## Klingon Numbers [3]

Klingon numbers are formed by combining a digit (0–9) with a power-of-ten suffix. [4, 5]

| Value [6, 7, 8, 9, 10] | Klingon Word         | Suffix            |
| ---------------------- | -------------------- | ----------------- |
| 0                      | pagh                 | —                 |
| 1                      | wa'                  | —                 |
| 10                     | wa'maH               | -maH (tens)       |
| 100                    | wa'vatlh             | -vatlh (hundreds) |
| 1,000                  | wa'SaD (or wa'SanID) | -SaD (thousands)  |
| 1,000,000              | wa''uy'              | -'uy' (millions)  |

-
- Compound Numbers: State the highest value first, followed by a space.
- Example: 42 is loSmaH cha' (four-tens two). [4]
- Ordinals: Add the suffix -DIch to the end of any cardinal number.
- Example: wa'DIch (first), cha'maH vaghDIch (25th). [8, 11]
- Counting vs. Numbering:
- Counting (Number before noun): cha' Duj (two ships).
  - Numbering (Number after noun): Duj cha' (Ship Number Two). [12]

## Dates and Time

Klingon time expressions usually appear at the beginning of a sentence. [2]

## Time Units

-
- Year: DIS
- Month: jar
- Day: jaj (dawn to dawn)
- Hour: rep
- Minute: tup
- Second: lup [2, 13]
-

## Formatting Dates [14]

While there is no single "official" date format (like MM/DD/YYYY) in the original [Klingon Dictionary](https://en.wikipedia.org/wiki/The_Klingon_Dictionary), common practice among linguists is to follow the largest-to-smallest hierarchy: [2, 15]

-
- Example: "On the 4th day of the 10th month of the year 2024"
- 2024 DIS jar wa'maH jaj loS
  - (Literally: 2024 year, 10th month, 4th day) [4]
-

## Time of Day [13]

Klingons typically measure a day in 16 hours, but for standard translation, they use the following terms: [13, 16]

-
- jajlo': Dawn
- pemjep: Midday
- ramjep: Midnight
- rep loS: 4:00 (Hour four) [6, 13]
